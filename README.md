# 🍦 KI DELICIA | Ice Cream Factory Website

Site institucional desenvolvido para uma fábrica de sorvetes com design moderno e sistema responsivo.  
Institutional website developed for an ice cream factory with a modern design and responsive system.

---

# 🇧🇷 Português

## 📋 Sobre o Projeto

Site institucional criado para apresentar a fábrica KI DELICIA de forma profissional, moderna e visualmente atrativa.

O projeto foi desenvolvido com foco em experiência do usuário, organização visual dos produtos e adaptação para diferentes dispositivos.

---

## ✨ Funcionalidades

✅ Layout moderno e intuitivo  
✅ Sistema totalmente responsivo  
✅ Adaptação automática para celulares, tablets e computadores  
✅ Navegação simples e agradável  
✅ Organização visual dos produtos  
✅ Interface profissional e otimizada  
✅ Estrutura preparada para futuras melhorias  
✅ Design focado em experiência do usuário
